# Assets

Drop the real hero photo here as `hero.jpg`. The site loads it at
`assets/hero.jpg` from `styles.css`.

Recommended specs:
- Landscape orientation, 1920×1080 or larger
- Subject can be a property, landscape, or interior — the navy overlay
  will tone it down so the cream headline stays readable.
- JPG, optimized to under ~400 KB if possible.
